# Hegelion Website

Static website for Hegelion built with React + Vite, matching the Shannon Labs design language.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The build output will be in the `dist/` directory, ready for Cloudflare Pages deployment.

## Deployment to Cloudflare Pages

1. Connect your repository to Cloudflare Pages
2. Set build command: `npm run build`
3. Set output directory: `dist`
4. Deploy!

The site will be automatically deployed on every push to the main branch.
